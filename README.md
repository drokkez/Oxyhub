# Doom на Python

Это упрощённая версия игры Doom, реализованная на Python с использованием библиотеки Pygame. Игра включает в себя 3D-рендеринг с помощью raycasting, систему передвижения игрока, врагов и стрельбу.

## Установка и запуск на Windows

### Требования
- Установленный Python 3.x
- Командная строка Windows (cmd)

### Установка

1. Откройте командную строку Windows (нажмите Win+R, введите `cmd`, затем Enter)

2. Перейдите в директорию, где вы хотите установить игру:
```
cd путь\к\директории
```

3. Скачайте или скопируйте файлы проекта в эту директорию

4. Установите зависимости, выполнив команду:
```
pip install -r requirements.txt
```

### Запуск

После установки зависимостей запустите игру командой:
```
python main.py
```

### Управление

- W / Стрелка вверх: движение вперёд
- S / Стрелка вниз: движение назад
- A: стрейф влево
- D: стрейф вправо
- Стрелки влево/вправо: поворот
- Пробел: выстрел

### Особенности

- Игра использует алгоритм raycasting для создания 3D-эффекта
- На экране отображается мини-карта с позициями игрока и врагов
- Враги преследуют игрока, когда он попадает в их зону обнаружения
